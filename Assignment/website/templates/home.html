Welcome!

The purpose of this program is to help manager your project!

Assuming your following the agile methodology.

-------------------------------------------------------------------------------

First of all, users will need to be created in order to create tasks.
The users page will be where users are created and deleted.
Additionly, once a sprint starts a burndown chart displaying individual contributions will appear on this page.
Please use the button below to navigater to the users page:
Users

Secondly, tasks will need to be created in order to start a sprint.
The tasks page will be where tasks are created, edited and deleted.
Please use the button below to navigater to the tasks page:

View Tasks

Lastly, the sprints page will be where to create and manage sprints.
Once a sprint' been created, tasks must be selected in order to start the sprint.
Note: One sprint can only be started/'in progress' at a time.
For "In Progress" sprints, log time option can be used to record time and Sprint Board can be used to progress tasks.
The program will automatically move the sprint into "Completed" once all the tasks have been completed.
Moreover, a option will be avaible to view the burndown chart of a sprint.

View Sprints

The navition bar can also be used to navigate to these pages
